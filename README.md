# Course project: MARKETPLACE API, iOS app edition.
This course project was created with a team: we created APIs for the client be able to buy products and the vendor to control the orders.
I was the one responsible for the client-side iOS application.

## This course project was developed with SwiftUI and Combine framework. Firebase Analytics and Crashlytics were used for logging as well.
